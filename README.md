# DESIKART – E-commerce Website

This project is developed as part of the CodSoft Internship.

## Live Demo
Frontend: [https://codsoft-desikart-parul.netlify.app](https://codsoft-desikart-parul.netlify.app)

## Features
- User Login & Signup
- Product Categories
- Add to Cart
- Quantity Update
- Protected Checkout Page
- Order Summary
- Payment Integration

## Technologies Used
- React.js
- React Router
- CSS
- JavaScript
- Axios
- MongoDB

## Deployment
Frontend is deployed on Netlify: [Live Demo](https://codsoft-desikart-parul.netlify.app)  
Backend is deployed on Render: [https://codsoft-2xsc.onrender.com](https://codsoft-2xsc.onrender.com)
